# AiNews 产品需求文档（简版）

## 1. 产品概述

**AiNews** 是一款面向银行工作人员的 AI 新闻专业分析系统。系统围绕"关键词"维度，对已收集的新闻数据进行 AI 质量评分、多维度统计分析和结构化报告生成，帮助银行从业者快速掌握行业政策动态、监管变化与市场情报。

**核心价值**：将分散的新闻信息转化为可评分、可对比、可导出的专业分析资产，辅助银行经营决策。

---

## 2. 系统功能架构

AiNews 采用"数据底座 + 双核心分析模块"的分层架构，各功能模块围绕统一的新闻数据库独立运作。

```
┌─────────────────────────────────────────────────────────────────────┐
│                         数据底座（Data Layer）                       │
├─────────────────────────────────────────────────────────────────────┤
│  新闻聚合（多源采集） → AI 智能评分（0-5分制） → 评分修改（人工校准）  │
└─────────────────────────────────────────────────────────────────────┘
                              │
        ┌─────────────────────┴─────────────────────┐
        ↓                                             ↓
┌─────────────────────────┐               ┌─────────────────────────┐
│   通用新闻分析模块        │               │    扬州公积金专栏        │
├─────────────────────────┤               ├─────────────────────────┤
│ • 关键词总览             │               │ • 周报政策对比            │
│ • 字数统计               │               │ • 地域政策浏览            │
│ • 周报生成               │               │ • 地区政策报告            │
└─────────────────────────┘               └─────────────────────────┘
                              │
                              ↓
┌─────────────────────────────────────────────────────────────────────┐
│                      系统配置与交付能力                              │
├─────────────────────────────────────────────────────────────────────┤
│  评分编辑 │ 配置管理 │ PDF 导出 │ Word 导出 │ 图片导出              │
└─────────────────────────────────────────────────────────────────────┘
```

## 3. 功能模块全景

### 3.1 周报生成（Report）⭐
基于关键词自动聚合周期内新闻，AI 生成结构化周报，支持多模型选择与深度定制。

- **周期智能设定**：默认日期范围为上周日至昨日，支持自定义调整
- **新闻聚合筛选**：按关键词拉取周期内新闻，支持评分过滤（全部/≥4分/≥3分）
- **内容摘要切换**：支持短摘要与完整原文两种内容模式切换
- **新闻手动遴选**：单条勾选/全选/取消全选，精确控制纳入周报的新闻范围
- **AI 模型选择**：DeepSeek R1（主模型）、KIMI K2、SiliconFlow 三模型切换
- **关键词特定提示词**：支持为不同关键词配置专属分析策略与 Prompt 模板
- **消息预览**：生成前可预览发送给 AI 的完整 Prompt，透明可控
- **流式生成**：SSE 实时推送 AI 生成内容，边生成边展示，支持显示推理过程
- **两轮修改**：初稿生成后，支持提交修改意见进行两轮迭代优化
- **多格式导出**：图片、PDF、Word 文档一键导出，PDF 由服务端 Playwright 渲染
- **断点重试**：生成失败时保留请求参数，支持一键重试

### 3.2 扬州公积金专栏 ⭐
面向住房公积金领域的专项政策追踪与分析模块，以扬州为基准，提供跨区域政策对比与深度分析能力。

- **周报政策对比**
  - 按周汇总扬州公积金及相关城市政策动态
  - 多期周报纵向对比，追踪政策变化轨迹
  - 扬州与其他城市横向对标，结构化展示同类型政策差异
  - AI 自动生成政策差异解读与趋势判断
  - 支持结构化报告解析与可视化卡片展示
  - 一键导出政策对比 PDF 报告与 JSON 数据

- **地域政策浏览**
  - 树形层级展示：全国 → 省份 → 地级市，支持折叠展开
  - 按地区筛选查看政策相关新闻，支持日期范围过滤
  - 默认定位扬州，快速聚焦目标城市
  - 实时统计各地区政策新闻数量与更新动态
  - 分页浏览新闻列表，支持查看原文链接

- **地区政策报告**
  - 多地区自由勾选（单地区或多地区组合）
  - 支持全国、省级、地市级任意混合选择
  - 智能匹配 Prompt 模板：单地区自动选单地区模板，多地区自动选多地区模板
  - 新闻预览与人工干预：生成前可预览各地区的政策新闻，手动排除/纳入特定条目
  - AI 自动生成专题分析报告，基于选定地区的新闻与政策文本
  - Prompt 版本管理：支持自定义、编辑、复制 Prompt 模板，设置默认版本
  - 报告生成后可导出 PDF，服务端渲染保障排版质量

### 3.3 评分编辑（Score Edit）
AI 评分后的人工校准环节，确保评分结果符合银行业务判断标准。
- 密码保护的管理员界面，前后端双重验证
- 单条/批量新闻评分调整，支持快速批量操作
- 实时同步更新至数据库，影响下游分析与报告
- 新闻内容预览与原文链接快速跳转

### 3.4 字数统计（Word Count）
内容规模监控与来源质量量化分析，辅助评估信息采集覆盖度。
- 90 天滚动数据统计，支持趋势对比
- 按关键词细分统计，掌握各业务线内容体量
- 评分分段字数分布，评估高质量内容占比
- 来源类型分析（自定义抓取 vs 微信渠道），监控渠道结构

### 3.5 配置管理（Config）
- LLM 模型切换与参数配置
- 系统/用户/修改三层 Prompt 模板编辑
- 关键词特定提示词 CRUD 管理
- 地区政策报告 Prompt 配置
- 配置热重载，无需重启服务

## 4. AI 能力体系

| 能力项 | 说明 |
|--------|------|
| **多模型调度** | DeepSeek R1（推理主力）、KIMI K2（长文本备用）、SiliconFlow（高可用备用） |
| **三层提示词** | System Prompt（角色定义）→ User Prompt（任务输入） |
| **关键词级定制** | 每个关键词可独立配置专属 Prompt，优先级高于默认模板 |
| **流式响应** | Server-Sent Events 实时输出，降低等待感知 |
| **自动重试** | AI 调用失败时自动重试（最多 3 次），保障稳定性 |

---

## 5. 导出与交付能力

- **PDF 导出**：服务端 Playwright 渲染，支持中文排版与分页
- **Word 导出**：原生 docx 格式生成，保留标题层级
- **图片导出**：html2canvas 前端渲染长图，适合移动端分享

---

## 6. 数据架构要点

- ** scored_news**：AI 评分新闻主表（标题、内容、链接、来源、评分、关键词、抓取日期、字数）
- **summary_news**：每日关键词摘要（按轮次存储多版本总结）
- **news_source_stats**：来源维度日统计表
- **policy_versions**：地区政策文本版本库
