# Google搜索API配置指南

## 概述

Test Gemini页面需要使用Google自定义搜索API来获取实时的搜索结果。本指南将帮助您完成相关配置。

## 配置步骤

### 1. 获取Google API密钥

1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 创建新项目或选择现有项目
3. 启用 "Custom Search API"
   - 在"API和服务" > "库"中搜索"Custom Search API"
   - 点击"启用"
4. 创建API密钥
   - 转到"API和服务" > "凭据"
   - 点击"创建凭据" > "API密钥"
   - 复制生成的API密钥

### 2. 创建自定义搜索引擎

1. 访问 [Google可编程搜索引擎](https://cse.google.com/cse/)
2. 点击"新增"或"创建自定义搜索引擎"
3. 配置搜索引擎：
   - **要搜索的网站**：输入 `*`（表示搜索整个互联网）
   - **语言**：选择"中文"
   - **搜索引擎名称**：例如"江苏省国资委新闻搜索"
4. 点击"创建"
5. 在控制面板中，点击"自定义"进行高级设置：
   - **搜索功能** > **图片搜索**：关闭
   - **搜索功能** > **安全搜索**：适中
   - **高级** > **搜索限制**：可设置为仅搜索中文页面
6. 复制"搜索引擎ID"（在"基本信息"标签中）

### 3. 环境变量配置

在项目根目录的`.env`文件中添加以下配置：

```bash
# Google搜索API配置
GOOGLE_API_KEY=your_google_api_key_here
GOOGLE_SEARCH_ENGINE_ID=your_custom_search_engine_id_here
```

**重要提示**：
- 将 `your_google_api_key_here` 替换为步骤1中获取的API密钥
- 将 `your_custom_search_engine_id_here` 替换为步骤2中获取的搜索引擎ID
- 确保`.env`文件不要提交到版本控制系统

### 4. API配额和限制

**免费配额**：
- 每天100次免费搜索请求
- 超出后按使用量收费

**付费计划**：
- $5美元/1000次请求（超出免费配额后）
- 每天最多10,000次请求

**请求限制**：
- 单次请求最多返回10个结果
- 支持分页获取更多结果

## 测试配置

### 方法1：使用测试脚本（推荐）

配置完成后，首先使用专门的测试脚本验证配置：

```bash
# 在项目根目录运行
node config/test-google-api.js
```

该脚本会：
- 检查环境变量是否正确设置
- 测试API密钥和搜索引擎ID是否有效
- 发起真实的搜索请求验证功能
- 提供详细的错误诊断信息

### 方法2：在应用中测试

脚本测试通过后，您可以：

1. 重启开发服务器：`npm run dev` 和后端服务器
2. 访问 `/test-gemini` 页面
3. 设置日期范围并点击"执行Google搜索"
4. 如果配置正确，应该能看到搜索结果

### 调试信息

如果遇到问题，Test Gemini页面会显示详细的调试信息：
- 点击"🔧 搜索调试信息"查看API调用详情
- 查看浏览器控制台获取更多错误信息
- 检查后端服务器日志获取完整的调试输出

## 常见问题

### Q: API密钥无效
**A**: 检查以下几点：
- API密钥是否正确复制
- 是否启用了Custom Search API
- 项目配额是否充足

### Q: 搜索引擎ID错误
**A**: 确保：
- 复制的是"搜索引擎ID"而不是其他ID
- 搜索引擎状态为"启用"

### Q: 搜索结果为空
**A**: 可能原因：
- 搜索关键词太具体
- 日期范围限制过严
- 搜索引擎配置限制了搜索范围

### Q: 超出API配额
**A**: 解决方案：
- 等待次日配额重置
- 升级到付费计划
- 优化搜索频率

## 注意事项

1. **API密钥安全**：
   - 不要将API密钥硬编码在前端代码中
   - 定期轮换API密钥
   - 为API密钥设置使用限制

2. **成本控制**：
   - 监控API使用量
   - 设置预算警报
   - 考虑缓存搜索结果

3. **搜索质量**：
   - 优化搜索关键词
   - 适当设置日期范围
   - 根据需要调整搜索引擎配置

## 高级配置（可选）

### 地域限制
在搜索请求中可以添加地域限制：
```javascript
// 限制中国地区结果
cr: 'countryCN'
```

### 语言限制
```javascript
// 限制中文结果
lr: 'lang_zh'
```

### 日期过滤
```javascript
// 日期范围过滤
dateRestrict: 'd:20240101..20240131'
```

这些配置已在代码中实现，无需额外设置。
